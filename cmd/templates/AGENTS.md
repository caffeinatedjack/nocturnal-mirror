
# Agent Instructions
This project uses nocturnal for specification management.

## Working on a Proposal
Use the nocturnal tools to get information on the current proposal, project and specifications. You can use the `nocturnal_todowrite` to manage the global todo list

## Rules
- You MUST read the projects rules using `nocturnal_rules` and follow them at all times.
- Follow the specification in `specification/current/specification.md`
- Update `specification/current/implementation.md` as tasks are completed
