Agent commands for reading and writing todo lists to TODO.md,
managing the active proposal context, and accessing documentation.

Commands:
    todowrite       Write tasks to TODO.md in the current directory
    todoread        Read and display TODO.md from the current directory
    current         Show the currently active proposal
    project         Show project rules and design
    specifications  Show completed specifications (alias: specs)
    docs            Search/list components in ~/.docs

Examples:
    nocturnal agent todowrite < todos.json
    nocturnal agent todoread
    nocturnal agent current
    nocturnal agent project
    nocturnal agent specs
    nocturnal agent docs list
    nocturnal agent docs search "component"
